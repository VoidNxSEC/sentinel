### Cloned Repository


### Local Development Override

```bash
# Test with local changes (no git push needed)
nix develop --override-input neoland path:../neoland

# Or switch to development flake
mv flake.nix flake.github.nix
mv flake.development.nix flake.nix

```

## 🧪 CI/CD Integration

### GitHub Actions (Automatic)

```yaml
# .github/workflows/nix-test.yml
name: Nix Integration Tests

on: [push, pull_request]

jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      
      - uses: DeterminateSystems/nix-installer-action@main
        with:
          extra-conf: |
            experimental-features = nix-command flakes
      
      - name: Run integration tests
        run: nix run .#test
```

The flake automatically uses GitHub URIs, so CI works out of the box!
